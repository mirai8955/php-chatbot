### Phase 2: 実装開始（1週間）

2. 採点ルール実装（GPT-5 YAML）
3. Level 1-3 問題作成（Haiku-4.5 + Sonnet-4.5）

### Phase 3: MVP リリース（2週間）
1. 基本的な評価エンジン
2. テンプレート型フィードバック
3. 進捗追跡システム

### Phase 4: AI 統合（3-4週間）
1. Claude API 統合
2. 動的問題生成
3. 高度なフィードバック
