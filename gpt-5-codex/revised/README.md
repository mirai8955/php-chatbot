# 改訂版ドキュメント概要

この `revised/` ディレクトリには、Monolog リポジトリを例にチームのコーディングスタイルを抽出するための改訂手法と、成果物テンプレートをまとめています。

## 背景
- 先行メモでは、一次情報の裏付けや構造化データが不足していた。
- 複数モデルからの評価フィードバックを踏まえ、再現性・定量性・AI 連携を意識した形で手法を再構築する必要がある。

## 収録ファイル
- `extraction_methodology.md` : 改訂手法の全体像と実行フロー。
- `style_rules_template.yaml` : 抽出結果を機械可読にまとめるためのテンプレート。
